
# StopWatch
### *By: James R. Brown*

> ## Description:  
>  A Static module build to be used to time code completion times.  These can be used to time 
>  a function or a piece of code to see how many seconds or part of a second the code takes to complete.
>  
>
>  
  
> ## Dependecies:
> Java 8  
> Java.time.LocalTime
